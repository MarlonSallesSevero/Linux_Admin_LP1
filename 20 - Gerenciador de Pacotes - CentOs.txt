**Gereciador de Pacotes - CentOs**

O repositorio do CentOs e diferente do Debian, aqui fica dentro de /etc/yum.repos.d/

Nesta pasta vamos ver varios arquivos .repo de arquivos para repositorios. 

cat arquivo.repo

[nome_repositorio]
name = descricao
mirrorList=caminho repositorios
#baseurl = 
gpgcheck=1 checagem gpg para seguranca do repo.
enabled=1 habilita/desabilita o repositorio 


*Gerenciadores de Pacotes no CentOs.

yum e dnf : Alto Nivel

yum repolist : traz quais repositorios estao habilitas no CentOs.

yum install epel-release : adiciona o repositorio da epel muito bom adicionar apos instalacao.

yum makecache : utilizado para atualizar com a lista de repo atualizados. Indexas e ver se tem coisa nova.

yum update : Pacotes que estao marcados para atualizacao serao atualizados.

yum list : vai listar todos os pacotes instalados na maquina e quais repo eles estao.

yum search pacote: realiza a pesquisa do pacote na maquina e ve se tem na maquina.

yum info pacote: Mostra um pequeno resumo para que serve o pacote e suas dependencias.

yum install pacote : realiza a instalacao dos pacotes, podemos fazer isso com mais de 1 pacote.

 rpm -q pacote : traz se o pacote esta instalado ou nao. 

 yum remove : realiza a remocao do pacote.

 yum download pacote : realiza a instalacao do pacote .rpm que e o pacote ja pre-compilado.



Exemplificando 

        debian         CentOs
    
    apt upgrade   =     yum update
    apt update    =     yum makecache


rpm : Baixo Nivel

rpm -i nome_pacote.rpm : realiza a instalacao do pacote.

rpm -qa : traz uma lista de todos pacotes instalados na maquina.

rpm -ql pacote : traz quais sao os arquivos que compoem esse pacote. E o local onde eles estao. 

rpm -qi : mostra o status do pacote.

rpm -qpil nome_pacote.rpm : Utilizado para a checagem do pacote.

    q : query / consulta
    p : packet / pacote consultado
    i : traz info do pacote
    l : traz a lista de pacotes.

rpm -ivh --percent --test: Traz mais detalhes do pacote testando o mesmo. Ele nao instala. 
    i : install
    v : verbose
    h = human 
    --percent : percentual da instalacao 
    --test : teste antes de instalar para ver se vai dar tudo certo. 

rpm -ivh --percent nome_pacote.rpm : Realiza a instalacao

rpm - Uvh nome_pacote.rpm : Faz um update do pacote instalado. / Instalacao

rpm -e nome_pacote : remocao do pacote. 

rpm -qf /bin/arq_bin : Utilizado para sabermos qual pacote esta presente o ls. Para outros pacotes, como se fosse uma engenharia reversa.

rpm -Va pacote : mostra os problemas de integridade do pacote. 
    V : varredura
    a : checa a integridade dos arquivos e as modificacoes realizadas.

*Muito usado por security. 

yum grouplist : Possibilidade de trabalhar com grupos de pacotes.

yum groupinfo Grupo_Pacote : traz alguns subgrupos _ obrigatorios e opcionais.

yum groupinstall 'Nome do Grupo' : A ser instalado 

yum groupremove 'Nome do Grupo' : A ser removido.
